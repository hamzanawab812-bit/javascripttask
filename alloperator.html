JavaScript Operators and Conditional Statements

Beginner guide. Simple language. Real examples.

1. Arithmetic Operators

Arithmetic operators work with numbers.

Just like math.

Common ones:

  • + Add numbers
  • - Subtract
  • * Multiply
  • / Divide
  • % Remainder

Example. Shopping bill.

let price = 1000;
let tax = 200;

let total = price + tax;

console.log(total);

Output: 1200

Example with strings.

let firstName = "Ali";
let lastName = "Khan";

console.log(firstName + " " + lastName);

Output: Ali Khan

Practice
Create two variables.
price = 500
discount = 100
Find the final price after discount.

2. Assignment Operators

Assignment operators store values in variables.

Example.

let price = 1000;

Now price contains 1000.

Shorter ways also exist.

  • += add and store
  • -= subtract and store

Example.

let wallet = 1000;

wallet += 500;

console.log(wallet);

Output: 1500

Real life example.

Your salary increases.

let salary = 30000;

salary += 5000;

console.log(salary);
Practice
Create a variable score = 50
Add 10 using +=
Print the result.

3. Comparison Operators

These operators compare values.

They return true or false.

  • == equal
  • === equal value and type
  • != not equal
  • < less than
  • > greater than
  • <= less or equal
  • >= greater or equal

Example. Exam result.

let marks = 80;

console.log(marks > 50);

Output: true

Difference between == and ===

This is important.

== compares values only.

=== compares value and data type.

Example.

console.log(5 == "5");

Output: true

Because JavaScript converts the string.

Now strict comparison.

console.log(5 === "5");

Output: false

Number is not the same type as string.

Developers usually prefer ===.

Practice
Create a variable age = 18
Check if age is greater than or equal to 18.

4. Logical Operators

Logical operators combine conditions.

  • && AND
  • || OR
  • ! NOT

Example. Login check.

let username = "admin";
let password = "1234";

console.log(username === "admin" && password === "1234");

Both conditions must be true.

Example using OR.

let day = "Sunday";

console.log(day === "Saturday" || day === "Sunday");

This checks weekend.

Practice
Create two variables.
isStudent = true
hasID = true
Check if both are true using &&.

5. Unary Operators

Unary means one value.

  • ++ increase by 1
  • -- decrease by 1

Example.

let count = 5;

count++;

console.log(count);

Output: 6

Example with visitors counter.

let visitors = 100;

visitors++;
Practice
Create a variable level = 1
Increase it using ++

6. Ternary Operator

This is a short way to write if else.

Structure:

condition ? value1 : value2

Example. Age check.

let age = 20;

let message = age >= 18 ? "Adult" : "Minor";

console.log(message);
Practice
Create marks = 40
Use ternary operator.
If marks >= 50 show "Pass"
Else show "Fail"

7. If Statement

If runs code when condition is true.

Example.

let age = 20;

if(age >= 18){
console.log("You can vote");
}

Use if when you only need one condition.

Practice
Create temperature = 35
If temperature > 30
Print "It is hot".

8. if else

Use when there are two possibilities.

Example. Login result.

let password = "1234";

if(password === "1234"){
console.log("Login success");
}else{
console.log("Wrong password");
}
Practice
Create age = 16
If age >= 18 print "Allowed"
Else print "Not allowed".

9. else if

Use when you have many conditions.

Example. Exam grading.

let marks = 75;

if(marks >= 90){
console.log("Grade A");
}
else if(marks >= 70){
console.log("Grade B");
}
else if(marks >= 50){
console.log("Grade C");
}
else{
console.log("Fail");
}
Practice
Create score = 85
Write conditions for A, B, C grades.

10. Switch Statement

Switch checks one variable against many values.

Good for menus or fixed options.

Example. Day of week.

let day = 2;

switch(day){
case 1:
console.log("Monday");
break;

case 2:
console.log("Tuesday");
break;

case 3:
console.log("Wednesday");
break;

default:
console.log("Invalid day");
}

Use switch when many exact values are checked.

Practice
Create a variable month = 1
Use switch to print January, February, March.
